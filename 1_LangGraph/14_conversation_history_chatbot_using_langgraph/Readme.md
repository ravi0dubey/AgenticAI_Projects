## Conversation_History Chatbot using Streamlit

Step 1 : Add a sidebar with title 
   1.1 : Generate dynamic thread_id and add it to the session
   1.2 : Display the thread_id in sidebar

Step 2 : Add a new Chat Button. On click of new chat Button
    2.1 : Open a new chat window
    2.2 : generate a new thread_id
    2.3 : Save it in session
    2.4 : reset message history

Step 3 : Create a list to store all thread_ids
Step 4 : Load all the thread ids in the sidebar
Step 5 : Convert side bar to clickable buttons
Step 6 : On click of a particular thread id load the particular conversation
